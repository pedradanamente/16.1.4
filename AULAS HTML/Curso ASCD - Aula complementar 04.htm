Sobre a questão 2 da aula 04
O prompt de comando do Windows permite o funcionamento de programas externos?
Resposta correta:Sim.


COMANDOS EXTERNOS

Um comando externo é realizado por outro programa, por exemplo o “PING” não é p prompt quem realiza a função mas sim o “ping.exe”
Os comandos externos padrões do Windows se encontra na pasta:
...\WINDOWS\system32





Sistema operacional MS-DOS

MS-DOS  é um antigo  sistema operacional, comprado pela Microsoft para ser usado na linha de computadores IBM PC.
Um sistema operacional como o MS-DOS e muitos outros tem a função de coordenar todo o funcionamento do computador oferecendo as aplicações ( planilhas, processadores de texto, etc )
O Prompt de comando visto na aula 04 é um interpretador de comandos baseado no MS-DOS, porém o prompt de comando não pe um sistema operacional.
Caso você não conheça  o antigo Sistema operacional MS-DOS, baixe DOS BOX que é um emulador do sistema operacional.
