para
Web Sites

 Importante!
 Andre, esta aula não tem por objetivo fazer você ser tornar um programador para web, mas você precisa conhecer quais são as linguagem mais utilizadas em web sites para poder analisar seu código fonte e procurar scripts maliciosos.
 A aula é destinada para iniciantes que não possuem conhecimentos em desenvolvimento para web.
 Serão apresentados apenas introduções as linguagens, não o ensino da linguagem em si, pois seria um outro assunto não referente ao nosso curso.


  HTML


Sem dúvida é a linguagem mais utilizada, também é a linguagem reconhecida pelos navegadores de Internet.
HyperText Markup Language, significa Linguagem de Marcação de Hipertexto.
Estrutura básica:
<html lang="pt">
   <head>
      <title>Título do Documento</title>
   </head>
   <body>
      texto,
      imagem,
      links,
      ...
   </body>
</html>


Como visualizar o código fonte de um site:
Internet Explorer: Pagina -> Exibir Codigo fonte
Firefox: Ctrl + U
Chrome: Ctrl + U




   JAVASCRIPT

A linguagem foi criada para atender, principalmente, às seguintes necessidades:
  • Validação de formulários no lado cliente (programa navegador);
  • Interação com a página.

Assim, foi feita como uma linguagem de script. JavaScript tem sintaxe semelhante à do Java, mas é totalmente diferente no conceito e no uso.
  Vocêpode inserir um trecho de javascript dentro de uma pagina HTML
Exemplo:
<html>
   <head>
      <title>Programa Olá Mundo</title>
   </head>
   <body>
      <script type="text/javascript">
         window.alert("Olá, Mundo!");
      </script>
   </body>
</html>

Ou deixar o script em um arquivo separado (.js)
src="arquivo_de_script.js"





   VBS

A versão VBScript implementada nos navegadores é muito semelhante em funções ao JavaScript.
Exemplo:
<html>
<body>

<script type="text/vbscript">
document.write("This is my first VBScript!")
</script>

</body>
</html>



   PHP

PHP é uma linguagem de programação de computadores interpretada, livre e muito utilizada para gerar conteúdo dinâmico na World Wide Web.
 Os navegadores não reconhecem um arquivo.php, somente os servidores que interpretam o código e emitem o código em HTML para a leitura dos navegadores.
Exemplo de código:
<?php
namespace Foo;
class Bar {
    public function Bar() {
        // treated as constructor in PHP 5.3.0-5.3.2
        // treated as regular method in PHP 5.3.3
    }
}
?>
