 SERVIDORES FTP
 
FTP é um acrônimo para File Transfer Protocol ou Protocolo de Transferência de Arquivos. O protocolo FTP permite a transferências de arquivos binários e arquivos texto com alta eficiência através de uma rede. É muito interessante para empresas que desejam compartilhar arquivos com os usuários na rede.

Utilização mais comum
 Servidores web que desejam disponibilizar áreas para usuários fazerem transferência de arquivos.
 O administrador do site geralmente usa FTP para enviar arquivos e pagina para o site.
Portanto alguém com acesso ao servidor, pode ter total controle dos arquivos do site.


Exemplos de servidores Online:
ftp.microsoft.com
ftp.xpg.com.br

Porta padrão:
A porta padrão para conexões em servidores FTP é a porta 21
