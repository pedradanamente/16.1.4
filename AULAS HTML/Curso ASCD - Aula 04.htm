PROMPT DE COMANDO 

Prompt de Comando é um interpretador de linha de comando no OS/2 e de sistemas baseados no Windows NT
O prompt é muito parecido com antigo MS-DOS.
O programa está localizado no diretório:

C:\WINDOWS\system32
(C: como Homedrive)

O executável é o cmd.exe

 

Assim como no WINDOWS EXPLORER onde utilizando o Mouse você pode criar pastas, mover arquivos, administrar o sistema, etc. Você também pode fazer tudo isto utilizando o Prompt de Comandos.

 


COMO ABRIR O PROMPT

No seu Windows XP entre em:
Iniciar -> Executar e Digite: cmd.exe


 

Ou pelo caminho:
Iniciar -> Todos os Programas -> Acessórios -> Prompt de Comandos

 

 

O WINDOWS EXPLORER recebe os comandos através do Mouse e do teclado, já o PROMPT é um interpretador de comandos de TEXTO, o mouse não é utilizado, apenas o teclado onde você digita os comandos.

 MANIPULANDO ARQUIVOS

 Você pode manipular arquivos utilizando apenas alguns comandos do prompt.

 RENOMEANDO ARQUIVOS:

Andre, vamos renomear um arquivo do Desktop, neste exemplo o arquivo: Novo(a) Documento de texto.txt
Utilize o comando REN
Vamos mudar o nome do arquivo para Arquivo_teste.txt

COMANDO:
REN "Novo(a) Documento de texto.txt" "Arquivo_teste.txt"

 

Observe agora os arquivos do Diretório:

COMANDO:
DIR

 

COPIANDO UM ARQUIVO

Vamos copiar o arquivo para o diretório meus Documentos

COMANDO:
COPY "Arquivo_teste.txt" C:\Documents and Settings\Lary Franciele\Meus documentos\Arquivo_teste.txt

 

Vamos Verificar a Pasta Meus Documentos

 

 VISUAL NO EXPLORER



 

LENDO O CONTEÚDO DE UM ARQUIVO DE TEXTO:

 
Utilize o comando TYPE para visualizar o conteúdo de um arquivo.
Vamos ver o conteúdo do arquivo Arquivo_teste.txt

COMANDO:
TYPE Arquivo_teste.txt

 

EXECUTANDO UM ARQUIVO

Utilize o comando START para executar um arquivo.
Vamos executar o conteúdo do arquivo Arquivo_teste.txt

COMANDO:
START Arquivo_teste.txt
É como se clicássemos no arquivo. ( O arquivo é aberto fora do Prompt ) (Não utilize aspas)
O arquivo tambem pode ser executado apenas digitando seu nome e estenção.



 

 

APAGANDO UM ARQUIVO

 
Para deletar um arquivo utilize o comando DEL
Vamos deletar o arquivo Arquivo_teste.txt

COMANDO:
DEL Arquivo_teste.txt
(Cuidado, pois o arquivo não vai para a lixeira)

 DIFERENÇA ENTRE COMANDO INTERNO E EXTERNO.

 O comando interno é um comando do próprio cmd.exe o próprio prompt que executa as funções, por exemplo o comando DIR, é o próprio CDM.exe quem lista os diretórios.

  VISUALIZANDO TODOS OS COMANDOS INTERNOS:

Até agora estes exemplo foram apenas uma introdução ao Prompt de comandos, pois são muitos comandos e funções que vamos aprender ao longo do curso.

Para visualizar Todos os comandos internos digite HELP no prompt ou clique aqui.


 COMANDOS EXTERNOS

Um comando externo é realizado por outro programa, por exemplo o PING não é p prompt quem realiza a função mas sim o ping.exe

Os comandos externos padrões do Windows se encontra na pasta:
...\WINDOWS\system32

 

 

 



CORES NO PROMPT DE COMANDO.

Voce pode alterar a cor do prompt utilizando o comando COLOR

COLOR <cor da fundo> <cor do fonte>
Código de cores em HEXADECIMAL:

 
0 = Preto
1 = Azul
2 = Verde
3 = Verde-água
4 = Vermelho
5 = Roxo
6 = Amarelo
7 = Branco
8 = Cinza
9 = Azul claro
A = Verde claro
B = Verde-água claro
C = Vermelho claro
D = Lilás
E = Amarelo claro
F = Branco brilhante

EXEMPLO:
Fundo Azul Fonte Verde:

Comando:
COLOR 1A
