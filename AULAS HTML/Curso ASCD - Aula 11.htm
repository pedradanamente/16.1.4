CARACTERES

Caractere (em informática) é o nome que se dá a cada um dos símbolos que se pode usar para produzir um programa ou arquivo de computador.
Por exemplo, um arquivo de texto:


 
Neste exemplo temos os seguintes caracteres (t,e,s,t,e,  ,c,u,r,s,o,  ,a,b,c,d).
O “espaço” também é um caractere.

Um caractere equivale à um BYTE.

Os bytes representam todas as letras (maiúsculas e minúsculas), sinais de pontuação, acentos, caracteres especiais e até informações que não podemos ver, mas que servem para comandar o computador e que podem inclusive ser enviados pelo teclado ou por outro dispositivo de entrada de dados e instruções.




BYTES e BITS

BIT
Os computadores "entendem" impulsos elétricos, positivos ou negativos, que são representados por 1 ou 0, respectivamente. A cada impulso elétrico, damos o nome de bit (BInary digiT).

BYTE
Um byte é um conjunto de 8 bits.
Os bytes representam todas as letras (maiúsculas e minúsculas), sinais de pontuação, acentos, caracteres especiais e até informações que não podemos ver, mas que servem para comandar o computador e que podem inclusive ser enviados pelo teclado ou por outro dispositivo de entrada de dados e instruções.

Conversões de Medidas

1 Byte = 8 Bit
1 Kilobyte = 1024 Bytes
1 Megabyte = 1048576 Bytes
1 Gigabyte = 1073741824 Bytes
1 Terabyte = 1099511627776 Bytes
1 Petabyte = 1125899906842624 Bytes
1 Exabyte = 1152921504606846976 Bytes
1 Zettabyte = 1180591620717411303424 Bytes
1 Yottabyte = 1208925819614629174706176 Bytes

Para converter uma unidade acima divide-se por 1024 e para converter uma unidade abaixo multiplica-se por 1024.
Exemplo: 2000 bytes para Kilobytes, uma unidade acima, é 2000/1024 que resulta em aproximadamente: 1.95 KB.

  MODO TEXTO E MODO BINARIO

  Dados no modo texto estão fundalmentalmente dentro da tabela ASCII, geralmente podemos visualizar e compreender os dados, é possivel até mesmo codificar um dado binário para o modo texto.
  Dados no modo binário utiliza os 256 caracteres em uma linguagem nao acessivel a primeira vista, e a ordem dos dados traduz apenas uma mensagem onde a maquina tem o papel de compreender.
