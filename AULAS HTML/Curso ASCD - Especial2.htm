Modo texto e Modo binário
Dados no modo texto estão fundalmentalmente dentro da tabela ASCII, geralmente podemos visualizar e compreender os dados, é possivel até mesmo codificar um dado binário para o modo texto.
Dados no modo binário utiliza os 256 caracteres em uma linguagem nao acessivel a primeira vista, e a ordem dos dados traduz apenas uma mensagem onde a maquina tem o papel de compreender.
