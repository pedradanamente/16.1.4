Navegadores

Um navegador, também conhecido pelos termos em inglês web browser ou simplesmente browser, é um programa de computador que habilita seus usuários a interagirem com documentos virtuais da Internet, também conhecidos como páginas da web, que podem ser escritas em linguagens como HTML,XHTML ou HTML5 com ou sem linguagens como o CSS e que estão hospedadas num servidor Web.

Arquivos
O navegador, tem a capacidade de ler vários tipos de arquivos, sendo nativo o processamento dos mais comuns (HTML, XML, JPEG, GIF, PNG, etc.),
Quando o navegador não consegue ler um arquivo o mesmo é forçado a download, por exemplo:
O  navegador consegue ler um arquivo tipo html
Clique no link: arquivos/teste.html
O  navegador consegue ler um arquivo tipo jpg
Clique no link: arquivos/teste.jpg
O  navegador não consegue ler um arquivo tipo rar
Clique no link: arquivos/teste.rar
Veja que o (.rar) foi forçado a Download em um diretório específico.

Plug-Ins
Alguns arquivos que o navegador nao consegue ler podem serem lidos por plug-ins instalados no proprio navegador, um exemplo são arquivos (.swf) Flash.
Linguagens
As linguarem mais populares interpretados pelos navegadores de internet são:
HTML, CSS, Javascrip, Vbscript
Importante: O navegador não interpreta scripts PHP, quando você abre uma pagina (.php) o que o navegador recebe são apenas códigos (HTML, CSS, Javascrip, Vbscript) o script PHP são codigos internos do servidor que irão gerar o conteúdo (HTML, CSS, Javascrip, Vbscript). O mesmo se aplica á ASP
