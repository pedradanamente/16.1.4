INVASÃO VIA BACKDOOR.

   Com a crescente democratização do acesso às tecnologias da Informação, a cada dia os riscos de segurança também aumentam na mesma proporção. O objetivo desta aula é mostrar como um simples arquivo mal intencionado pode causar sérios danos principalmente se tratando de sua privacidade na Internet.
Alem de mostrar os riscos de Backdoors também vamos estudar formas de defesa de uma forma técnica.
Em nosso tutorial vamos utilizar um único software tanto para pen test como para proteção. Nosso software é destinado exclusivamente para estudos e contem uma série de proteções para não ser usado em situações ilícitas. Nosso objetivo é estimular os estudos de segurança de uma forma segura e ética.

 



BACKDOORS

 Backdoors são programas que instalam um ambiente de serviço em um computador, tornando-o acessível à distância, permitindo o controle remoto da máquina sem que o usuário saiba. Isto significa que um computador infectado para a ser controlado por outra pessoa remotamente e de uma forma totalmente oculta.

 Diferença entre Backdoors e Spywares:
Backdoors: Apenas abre uma porta para acesso remoto.
Spywares: Capturas dados pessoais do usuário e envia para o “Cracker” geralmente via email.


 Andre, um Backdoor pode ser muito mais perigoso que um spyware pois os Spywares são espalhados em formas de worm e o controlador não escolhe seu alvo, ele apenas deixa o worm se espalhar na rede e fica esperando os dados de computadores infectados, consequentemente seu código se torna publico e rapidamente as empresas de antivírus criam vacinas e você obtém a defesa sem mesmo saber.

 Com os Backdoors a maioria das infecções funcionam de forma diferente, o controlador escolhe seu alvo, planeja seu ataque, camufla a integridade do arquivo,etc.

Isto pode ser um grande risco principalmente para Empresas, Ambientes corporativos pois os alvos são planejados.

 Um Backdoor pode ficar oculto por muitos anos com nenhum antivírus sabendo de sua existência ( Com exceção de trojans conhecidos ).

 Nossa política de segurança:

Você configurou uma senha, portanto somente você pode ter acesso ao sistema.

  CMD.exe
  Próprio Backdoor

 O Backdoor Insecuitynet 3.X possui vários comandos a nível SHELL e também abre o CMD.exe



 
ACESSANDO A SHELL CMD.EXE

Para acessar o CMD.exe basta entrar em algum diretório (Desktop, Meus Documentos ou Raiz)
Vamos entrar no Desktop ( Digite 1 e tecle enter )


 

 Observe que agora você tem controle total sobre o computador, você poderá executar, deletar, visualizar, mover, copiar arquivos como se você estivesse no Prompt de comando do computador alvo.
 Para sair da SHELL CMD.exe e voltar ao Menu (Shell do Backdoor) , digite: exit

 



REMOVENDO O BACKDOOR

Você tem duas opções:
Remoção Local Digite: /quit para desconectar e entre em:  Iniciar -> defesa -> Vacina
Remoção Remota No menu do Backdoor digite o comando: REMV
