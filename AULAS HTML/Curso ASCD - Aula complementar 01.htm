Aula Complementar


Sobre o exercício prático:

Você pode fazer o exercício de conexão no servidor pop utilizando apenas o prompt de comando.
No seu Windows clique em Iniciar -> executar:



Digite cmd



Veja que o Prompt de Comando foi aberto.



Para conectar no servidor vamos usar o comando TELNET

A sintaxe completa do comando será
telnet ip porta

Por exemplo:
Se quizermos conecat no IP 127.0.0.1 na porta 50, o comando ficaria:
telnet 127.0.0.1 50



Na aula 01 temos o IP e porta do servidor de email.

Dica:
Crie uma conta de email bol e faça a conexão de sua caixa (POP) apenas utilizando o prompt de comando.
