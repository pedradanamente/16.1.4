PARA QUE SERVE AS PORTAS:

Andre, imagine uma empresa com vários setores e um único número de telefone,e precisa ter acessos diretos os setores de forma externa. A solução seria usar um PABX com vários ramais.Exemplo:
Ramal 01 = Recepção
Ramal 02 = Diretoria
Ramal 03 = Sala de reunião
Ramal 04 = Controle de estoque

No computador é a mesma coisa, existe apenas um endereço IP, mas vários programas e serviços, por exemplo: Navegador, Messenger, Jogos online, etc.). Cada serviço deve ter sua porta específica assim com apenas um endereço IP você pode acessar diferentes serviços do computador.

PORT SCANNERS

O Port Scan é ferramenta indispensável para sua própria defesa e também para verificar possíveis vulnerabilidades em servidores remotos.
A ferramenta é utilizada para scannear um computador a procura de portas abertas.
Localizando uma porta aberta podemos saber qual serviço está ativo na porta como por exemplo (FTP, HTTP, Telnet, SMTP etc )

FAIXA DE PORTAS

Faixa de portas é um intervalo entre duas portas, exemplo a faixa de portas entre 22 e 29:

22, 23, 24, 25, 26, 27, 28 e 29.

Caso você queira, você pode fazer um Scan apenas em uma determinada faixa.

Inicio -> Scan e Monitoramento -> Port Scan Faixa

Neste exemplo vamos fazer um scan de faixa entre as potas 75 e 85

 


As portas TCP mais usadas (também chamadas de "well known ports") são as portas de 0 a 1023, que são reservadas para serviços mais conhecidos.


 

Como Scanear seu próprio computador.

Basta fazer um Scan no IP: 127.0.0.1

Este IP é o endereço do localhost, sua maquina local.

 




Uma porta aberta significa vulnerabilidade?

Andre, uma porta aberta indica que existe uma possibilidade de conexão em um determinado serviço.

Por exemplo, servidor www.microsoft.com tem a porta 80 aberta, pois seu navegador precisa se conectar no servidor para o servidor poder enviar o conteúdo do site, todo o processo de conexão é feito através da porta 80, esta porta precisa estar aberta para que o site esteja no ar. Isto não é uma vulnerabilidade.

 Porem você pode listar todas as portas abertas de um determinado servidor e explorar o que cada porta faz, no exemplo prático exploramos a porta 21 e o servidor FTP aceitou um login anônimo (neste caso proposital).

* Vários vírus de computador abrem portas para acessos remotos onde você pode ter acesso total ao sistema.

Você pode identificar servidores desatualizados sujeitos a exploits.
